## Investment Risk & Volatility Monitor (AlphaPulse)
# Project Overview
This project was developed as part of an internship program. It focuses on analyzing portfolio market risk, measuring volatility, and calculating Value at Risk (VaR) to support data-driven investment decision-making. The system provides real-time portfolio insights, risk forecasting, and correlation analysis to help optimize diversification strategies.Investment Risk & Volatility Monitor (AlphaPulse)

# Tools & Technologies
- Python
- yfinance
- NumPy
- PowerBI
- SQL (Optional for storage)

# Key Features
- 📊 Real-time stock data collection using yfinance 
- 📈 Daily log returns and portfolio performance analysis
- 🎯 Value at Risk (VaR) calculation at 95% and 99% confidence levels
- 🔁 Monte Carlo simulation (10,000+ runs) for 1-year portfolio forecasting
- 🔥 Dynamic correlation heatmaps for diversification insights
- 📉 30-day rolling volatility tracking to measure market uncertainty
- 📊 Interactive risk dashboard built using PowerBI

# Development Workflow
Market Data → Data Cleaning → Log Return Calculation → Covariance Matrix → Monte Carlo Simulation → VaR Calculation → PowerBI Dashboard
